Starter kit overview
Overview

Each layout contains a set of blank pages called "Starter kit" (or Seed project). This kit may be very useful for developers who will work on integration. Each page includes a basic markup - navbar, sidebar, content area and a footer. The functionality is limited by default to necessary elements only - jquery and bootstrap JS libraries; bootstrap, layout, pages, helpers and core CSS files compiled from LESS.

Screenshot

Usage

Usage of a Starter kit is very simple - add necessary library, extension or plugin path, copy necessary code from the page .js files located in assets/js/demo_pages/ folder to a new JS file, add a path to this file in your page, re-compile LESS with styling of this plugin and use it. As simple as that.

All unnecessary component styles are commented in components.less file, so if you need any additional styles, just uncomment the path to LESS file and re-compile it. This method allows you to use only those components styles that are actually needed, no extra stuff.

All template pages include only those JS files, that are needed on the page. So if you need, for example, some specific functinality of Datatables, just open this page and a .js file that comes after app.js path, copy the code and paste it to your file. This helps to avoid loading of unnecessary and unused files.

Main files
Layout 1 and Layout 2

First and second layouts contain 14 blank pages. All files include a simple demo content - panel, text with headings and paragraphs, table, form and content grid samples. By default, navbar has a set of basic components - user dropdown menu, text and icon links. Sidebar has a single and multiple levels navigation with divider and section header. Starter kit of the first and second layouts contain all available layout types and options:

What Description
Navigation
horizontal_nav.html.html Horizontal multi level navigation with mega menu and state saving feature
Columns
1_col.html Full width page template
2_col.html 2 columns page template - main sidebar and content area
3_col_double.html 3 columns page layout - with main and opposite sidebars
3_col_dual.html 3 columns page layout - with main and secondary sidebars
4_col.html 4 columns page layout - with 3 sidebars and content area
Detached sidebars
detached_left.html Left aligned detached sidebar
detached_right.html Right aligned detached sidebar
detached_sticky.html Sticky detached sidebar, with custom scrollbar
Layout options
layout_boxed.html Boxed page layout
layout_fixed.html Layout with fixed sidebar and navbar, with custom scrollbar
layout_navbar_fixed_main.html Multiple navbars - fixed main navbar
layout_navbar_fixed_secondary.html Multiple navbars - fixed secondary navbar
layout_navbar_fixed_both.html Multiple navbars - both main and secondary are fixed
